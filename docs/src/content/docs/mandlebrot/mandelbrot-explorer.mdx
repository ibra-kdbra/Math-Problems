## Mathematical Foundation

<TextBlock size="base" maxWidth="3xl" isProse invertInDark>
The Mandelbrot set is defined by the iterative complex function:

<Math formula="z_{n+1} = z_n^2 + c" displayMode={true} />

Where:
- <Math formula="z_0 = 0" displayMode={false} />
- <Math formula="c" displayMode={false} /> is a complex constant representing a point in the complex plane
- A point <Math formula="c" displayMode={false} /> belongs to the Mandelbrot set if the sequence remains bounded
</TextBlock>

### Convergence Criteria

<TextBlock size="base" maxWidth="3xl" isProse invertInDark>
The algorithm tests for convergence using:
<Math formula="|z_n| \leq 2" displayMode={true} />

If the magnitude exceeds 2, the sequence diverges and the point is not in the set. The number of iterations before divergence determines the coloring scheme.
</TextBlock>

### Complex Arithmetic Optimization

<TextBlock size="base" maxWidth="3xl" isProse invertInDark>
```python
def mandelbrot_iteration(c_real, c_imag, max_iter):
    z_real = 0.0
    z_imag = 0.0
    
    for i in range(max_iter):
        z_real_squared = z_real * z_real
        z_imag_squared = z_imag * z_imag
        
        if z_real_squared + z_imag_squared > 4.0:
            return i
            
        z_imag = 2.0 * z_real * z_imag + c_imag
        z_real = z_real_squared - z_imag_squared + c_real
    
    return max_iter
```
</TextBlock>
