# Mandelbrot Coloring

SFML-based Mandelbrot viewer program.

![img](/imgs/mandlebrot_coloring/mandelbrot.png) 
![img](/imgs/mandlebrot_coloring/zoomed.png)

## Requirements
 * C++17
 * CMake 3.22

## Building & Running

```shell
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release --target run
```

### Controls

<Aside type='note'>Usage</Aside>

| Action            | Control         |
| ----------------- | --------------- |
| Go to point       | Click           |
| Zoom              | Scroll (or W/S) |
| Pan               | Arrow keys      |
| Change iterations | [ and ]         |
| Reset view        | R               |
