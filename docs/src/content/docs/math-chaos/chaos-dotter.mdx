# Chaos Visualization

<TextBlock size="lg" maxWidth="3xl" isProse={true} invertInDark={true}>
This HTML project creates an interactive visualization of a mathematical concept known as the **Chaos Game**.
It generates fractal patterns by repeatedly plotting a point that moves a certain percentage toward randomly chosen vertices of a polygon.
</TextBlock>

## How It Works

<Aside type="tip">
For optimal results, experiment with different vertex counts and ratios. Try the golden ratio (61.8%) for particularly beautiful patterns!
</Aside>

<Steps>
  1. **Initialize Audio**
     Audio playback begins automatically with `Gesaffelstei-Aleph.mp3` looped in the background.

  2. **Setup Parameters**
     - Choose polygon vertex count (â‰¥3)
     - Set movement ratio (e.g., 0.618 for golden ratio)

  3. **Canvas Preparation**
     - Position vertices equidistantly on a circle
     - Initialize starting point randomly
     - Clear previous drawings

  4. **Chaos Generation**
     - Select random vertex
     - Move current point toward vertex by specified ratio
     - Plot new point
     - Repeat thousands of times for pattern emergence
</Steps>
