## Setup your development

Learn how to update your docs locally and deploy them to the public.

1. Clone the repository

<CodeBlock
  code={`git clone https://github.com/ibra-kdbra/Math_Problems`}
  lang='bash'
/>

2.  Navigate to the project directory

<CodeBlock code={`cd docs`} lang='bash' />

3. Install the dependencies

<CodeBlock code={`npm install`} lang='bash' />

4. Start your development server

<CodeBlock code={`npm run dev`} lang='bash' />

Once the server starts, you can view your documentation site at `http://localhost:4321`.
